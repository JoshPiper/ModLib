    ArgError([funcName], argNum, expected, got)
    Create an argument error for the given function.

    Parameters:

    • funcName string or number If nil, the calling function's name, if number, the name of the function 2 + n in the sdtack, if string, that is used directly. (optional)
    • argNum number Which argument was incorrectly passed.
    • expected string The expected type.
    • got The recieved value.

    Returns:

      string The formatted argument error.
